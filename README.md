<li>How to Build and Work;</li>

<br>

You should be able to enter the following commands into the Freebsd SSH partition.

<br>
> > cd /root<br>
wget https://raw.github.com/vivoxy/freebsdscript/master/vivoxy.sh<br>
sh vivoxy.sh
